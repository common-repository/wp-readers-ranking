The Readers Ranking Wordpress Plugin. WordPress读者墙插件

== Description ==

Quickly Create A Readers Wall Showing The Rank of Comments Author On Blog.

快速创建一个读者墙，展示博客评论排行。

相比网络上已有插件，本插件具有以下插件特色：

    1、可限制不同的时间段，比如每周、每月、每年的评论排行，而且3种时间段排行可以任意组合，亦可同时存在；    
    2、可在后台设置全自动加载模式，并且可以选择展示的前后位置；    
    3、可为博客开启Gravatar头像加速，解决头像被墙问题;
    4、初级响应式设计，并且可以自定义CSS样式。

详情请浏览：[ http://zhangge.net/4756.html ]( http://zhangge.net/4756.html )

== Installation ==

可以通过以下两种方法的其中一种来安装WP Readers Ranking 插件：

1. 将下载的文件解压缩，然后将`wp-readers-ranking`文件夹 上传到 `/wp-content/plugins/`目录，在插件后台启用

2. 直接在后台-安装插件，搜索'wp readers ranking'，按照提示安装启用

Ps：首次安装，请先进入设置界面进行更详细的显示设置
